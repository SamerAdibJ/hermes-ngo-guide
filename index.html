📋 What You Need Before Starting

Really simple — just these four things. No experience required.

  • 💻 A computer — Windows 10/11 or macOS 12+. Any standard laptop works.
  • 📶 Internet connection — the AI runs in the cloud, so you need to be online to use it.
  • 🔑 A DeepSeek API key — free to create, costs a few dollars per month. We'll walk through getting one.
  • ⏱️ 5 minutes — that's how long the whole setup takes.

🔑 Step 1 — Get Your DeepSeek API Key

An API key is like a password that lets Hermes talk to DeepSeek's AI models. You only do this once.

1

Create a DeepSeek Account

Go to platform.deepseek.com/signup and sign up with your email or Google account.

2

Create an API Key

Once logged in:

1. Go to API Keys in the left sidebar

2. Click "Create API Key"

3. Give it a name like Hermes Desktop

4. Copy the key — it looks like sk-xxxxxxxxxxxxxxxx

⚠️ Copy it now — you won't see it again after you leave the page!

3

Add Credit (Optional but Recommended)

DeepSeek gives you a small amount of free credit to start. When that runs out, go to Billing → Top Up and add $5–10. That will last weeks of regular use.

DeepSeek pricing (June 2026): V4 Flash (best value): $0.44 /M input tokens → ~$0.30–0.80/day for NGO work V4 Pro (strongest): $1.74 /M input tokens → for complex analysis DeepSeek Chat (V3): $0.14 /M input tokens → cheapest, but being phased out DeepSeek R1 (reasoning): $0.55 /M input tokens → for step-by-step logic
✅ Done! Keep that key safe — treat it like a password. Never share it or paste it in public forums. You'll paste it into Hermes Desktop next.

🖥️ Step 2 — Install Hermes Desktop

This is the app you'll talk to. Install it like any other program — no terminal, no code.

1

Download the Installer

Go to hermes-agent.nousresearch.com and click "DOWNLOAD FOR WINDOWS" (or macOS if you're on a Mac).

⬇️ Download Hermes Desktop

The download is ~100MB — it should take less than a minute on a normal connection.

2

Run the Installer

Open the downloaded file and follow the prompts. It's the same as installing Zoom, Slack, or any other desktop app. The installer handles everything — including all dependencies.

3

Open Hermes Desktop

Once installed, launch Hermes Desktop from your Start Menu (Windows) or Applications folder (macOS).

You'll see a clean chat window. If it asks you to set up an account, click "Skip" or "Use API Key" — we'll connect it to DeepSeek next.

💡 What about the command line? You don't need it. Hermes Desktop gives you a full chat interface without ever opening a terminal. But if you get curious later, the CLI is installed too — just open Command Prompt and type hermes.

🔗 Step 3 — Add DeepSeek & Start Chatting

One paste of your API key, and Hermes is connected to DeepSeek's AI.

1

Open Settings in Hermes Desktop

In the Hermes Desktop app, click on your profile icon or the ⚙️ Settings gear icon (usually bottom-left or top-right corner of the window).

2

Select a Provider

Go to the Model / Provider section. Choose DeepSeek from the list of providers. If you don't see DeepSeek listed, choose Custom and enter:

Provider URL: https://api.deepseek.com/v1 Model name: deepseek-v4-flash
3

Paste Your API Key

In the API Key field, paste the key you copied earlier (sk-xxxxxxxxxxxxxxxx). Click Save or Connect.

✅ That's it! Hermes is now talking to DeepSeek. No command line, no configuration files, no technical steps.
4

Test It

Type your first message in the chat:

Hello! Who are you and what can you help me with?

If you get a reply — congratulations! 🎉 Your AI assistant is ready.

⚠️ Important note on model aliases: DeepSeek sometimes silently upgrades the deepseek-chat alias to a newer, more expensive model on their backend. To avoid surprises, use the explicit model name instead. We recommend deepseek-v4-flash — it's the best value and you'll know exactly what you're paying for. Check your actual usage at platform.deepseek.com/usage.

🧠 Memory — So Hermes Never Forgets

This is what makes Hermes different from ChatGPT or Claude: it remembers your name, preferences, past work, and learned facts across sessions. And it's all stored locally on your machine, not in the cloud.

📝

Built-in Memory (Default)

Hermes saves key facts as notes and reads them at the start of every session. Simple and effective — remembers your name, preferences, and important details. ~2,200 character limit, enough for the essentials.

🧬

Holographic Memory (Advanced)

Uses a local database with entity resolution, trust scoring, and search. No character limit, remembers structured facts, and can reason across multiple entities. Runs entirely locally. You can enable it in Settings.

Using Memory Day-to-Day

In the Hermes Desktop chat, just tell Hermes what to remember. It works like natural conversation:

"Remember that my name is Maher and I work at FES NGO"
"Our donor prefers email over phone calls"
"The quarterly report template is in our shared drive"
"What did I ask you about yesterday?"

Enable Holographic Memory (Optional)

1

Go to Settings → Memory

In Hermes Desktop, open Settings and find the Memory section.

2

Switch Provider to Holographic

In the memory provider dropdown, select Holographic instead of "Built-in". Save the setting, then restart Hermes Desktop for the change to take effect.

3

Test It

Tell Hermes something, close the app, reopen it, and ask about it. It will remember.

How Memory Works

Built-in memory stores simple text notes. Holographic Memory stores structured facts with entity resolution — it knows that "Maher at FES" and "the person who asked about the report" are the same entity. It uses trust scoring to learn which facts are reliable. It can even reason across entities: "What does Maher know about our donors?"

Feature Built-in Memory Holographic Memory
Storage ~2,200 character limit Unlimited SQLite database
Search Keyword match Semantic + keyword + entity resolution
Persistence Across sessions Across sessions + profiles
Confidence All facts equal Trust scoring per fact
Stored where? ✅ Your machine ✅ Your machine (local SQLite)
How to enable Default (always on) Settings → Memory → Holographic
🔑 Why this matters for your work: With memory enabled, Hermes can remember everything about your NGO work — donor preferences, reporting templates, past meeting notes, communication styles for each partner, and recurring tasks. It's like having a colleague who remembers everything you've ever discussed.

🎯 Which Provider & Model Should You Use?

You have options for how to connect. Here's what we recommend.

✅ Provider Options

Hermes supports DeepSeek natively. You can also use DeepSeek models through other providers if you prefer one key for multiple models.

Provider Setup Why Choose It
🌐 DeepSeek Direct Sign up → API key → paste in Hermes ⭐ Recommended. Lowest cost, direct access, fastest responses.
🎯 OpenRouter Sign up → API key → choose "DeepSeek" model One key gives you access to many models. Useful if you want to switch between DeepSeek, Claude, GPT, etc. without changing keys.
🤖 Together AI Sign up → API key → custom endpoint in Hermes Fast hosting for DeepSeek and open-source models. Good alternative if DeepSeek Direct is slow in your region.
🏆 Our recommendation: Start with DeepSeek Direct using model deepseek-v4-flash. Best value, simplest setup. If you want access to other models (Claude, GPT) from the same interface, switch to OpenRouter.

🔧 Troubleshooting

Common issues and how to fix them — no technical knowledge needed.

🔑

"Invalid API key" or "Auth failed"

Your key is wrong or expired. Go back to platform.deepseek.com/api-keys, generate a new one, and paste it in Hermes Settings → Model → API Key.

💰

"Insufficient balance"

You've run out of credits. Go to DeepSeek Platform → Billing → Top Up. Adding $5 usually lasts a month for daily use.

🌐

No response / "Connection failed"

Check your internet connection. If you're at an office with a firewall, you may need to ask your IT team to whitelist api.deepseek.com.

💸

Costs higher than expected

DeepSeek may have upgraded your model alias. Set the explicit model to deepseek-v4-flash in Settings → Model. Check usage at platform.deepseek.com/usage.

🐢

Slow responses

Usually a network issue. Check your internet speed. If it's persistent, try connecting via OpenRouter instead (sign up, get a key, use it as your provider in Hermes).

❓

Hermes Desktop won't open / crashes

Restart your computer and try again. If it still doesn't work, uninstall and reinstall from hermes-agent.nousresearch.com.
