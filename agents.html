🎯 How This Works

Simple rules for our agent-building journey.

🛠️

Build & Document

Every time you create a Hermes agent you use regularly, add it here. Name it, describe it, show the prompt.

🏅

Friendly Competition

Each agent = 1 point. Bragging rights to whoever is ahead. But the real win is the growing library you both share.

📚

Borrow & Improve

See what the other built? Fork the idea, make it your own. The collection evolves as both of you level up.

🤝

Collaborations

Build an agent together? Gets a special badge and counts for both on the scoreboard. Double the fun.

💡 How to add an agent: Edit this page on GitHub → add a new .agent block under your section → update the score numbers → push. Or just send me the details and I'll do it for you. Simple as that.
